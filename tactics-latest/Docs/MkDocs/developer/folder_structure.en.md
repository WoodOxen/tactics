# Folder Structure

The Unity project of Tactics locates at `./Tactics`. You have to select this directory to import project into a Unity Editor.

## Asset Path Structure

```shell
Tactics/Assets
├── AddressableAssetsData
├── Art
│   ├── Materials
│   ├── Models
│   ├── Shaders
│   ├── Sprites
│   └── Textures
├── Audio
├── Editor
├── Locales
├── Plugins
├── Prefabs
├── Sandbox
├── Scenes
├── Scripts
├── TextMesh Pro
└── ThirdParty
```
