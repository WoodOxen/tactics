# 文件夹结构

Tactics的Unity项目位于`./Tactics`，导入到Unity编辑器时，请选中对应的文件夹。

## 资源目录结构介绍

在Unity编辑器中，与开发者有关的目录是`./Tactics/Assets`。其基本结构如下：

```shell
Tactics/Assets
├── AddressableAssetsData
├── Art
│   ├── Materials
│   ├── Models
│   ├── Shaders
│   ├── Sprites
│   └── Textures
├── Audio
├── Editor
├── Gizmos
├── Locales
├── Plugins
├── Prefabs
├── Sandbox
├── Scenes
├── Scripts
├── TextMesh Pro
└── ThirdParty
```
